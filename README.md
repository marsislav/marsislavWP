# marsislav

A custom WordPress theme built on a solid foundation.

## Features

* Responsive design with mobile-first approach
* Custom mobile slide-in menu
* Customizable footer via WordPress Customizer (1 or 2 columns, dynamic copyright text)
* Clean, semantic HTML

## Installation

1. Upload the `marsislav` folder to `/wp-content/themes/`
2. Activate the theme in **Appearance → Themes**
3. Customize via **Appearance → Customize**
